# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript.
It showcases my profile, skills, projects, and contact information in a modern and user-friendly design.

## Features

* Responsive navigation bar
* Hero section with profile photo
* Skills section with animated progress bars
* Projects showcase
* Contact form
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
docs/
├── index.html
├── style.css
├── script.js
└── GPIC.jpeg
```

## How to Run

### Method 1: Using VS Code

1. Open the project folder in Visual Studio Code.
2. Open `index.html`.
3. Right-click and select **Open with Live Server** (if Live Server extension is installed).

### Method 2: Using a Browser

1. Open the `docs` folder.
2. Double-click `index.html` to launch the website in your browser.
